üì• Document Indexing Flow

Step 1: Load documents from docs/ folder (PDF, TXT, DOCX)
Step 2: Split into chunks using RecursiveCharacterTextSplitter
Step 3: Remove duplicate chunks (BLAKE2b hashing)
Step 4: Generate embeddings with HuggingFace multilingual-e5-base
Step 5: Build and save FAISS index locally

üí¨ Question Answering Flow

Step 1: User enters question via Streamlit chat interface
Step 2: RetrievalQA chain processes the question
Step 3: FAISS retrieves top-k most relevant chunks (similarity or MMR)
Step 4: Compose prompt with context from retrieved chunks
Step 5: Send to Ollama (mistral:7b) for answer generation
Step 6: Return answer with source citations
Step 7: Display answer in UI with expandable sources
